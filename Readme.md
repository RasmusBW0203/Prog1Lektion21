### Opgave 1

Kik på koden i main metoden i opgave 1. Koden er ikke særlig robust i 
forhold til brugerinput. Der er to exceptions der kan blive kastet, hvis ikke brugeren 
indtaster korrekt.

passende besked til brugeren i konsollen.
